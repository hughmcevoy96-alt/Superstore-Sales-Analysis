# Superstore E-commerce Data Analysis

## Description of Project

A data warehouse for an American online "Superstore" is designed using a synthetic e-commerce dataset and is implemented in SQL Server. It is followed by a full visualisation and analysis of the data using interactive Power BI dashboards accompanied by a written report, focusing on providing actionable business insights. A star-schema dimensional modelling approach is taken to the design of the warehouse to ensure correct structure and integrity of the data. Modifications were made to this design in Power BI to create a semantic model that optimized performance within the business analytics tool. Three interactive dashboards were created along with a full written report containing the insights of the analysis, focusing on trends in revenue, profitability of individual products and categories, customer segmentation and shipping performance of the Superstore.

## Technology Stack

- SQL Server
- T-SQL
- Power Query
- Power BI
- DAX

## Data Model

### Data Warehouse Design

![Data Warehouse Design](Data%20Warehouse%20Design%20+%20Semantic%20Model/Data%20Warehouse%20Design.png)

DDL and DML SQL was used to create and populate the star-schema data warehouse that was designed from the raw data table. Surrogate primary and foreign keys were used to create active relationships between the fact and dimension tables of the model, ensuring full integrity of the data.

### Power BI Semantic Model

![Semantic Model](Data%20Warehouse%20Design%20+%20Semantic%20Model/Power%20BI%20Semantic%20Model.png)

This model was refined in Power BI using Power Query to create a semantic model that would optimize efficiency within the business analytics tool. For example, a second date dimension table with an active relationship to the fact table was added to facilitate the creation of a measure which calculates the number of days between an order being received and the order being shipped.

## Dashboards

![Revenue Breakdown - Dashboard](Dashboards/Revenue%20Breakdown%20-%20Dashboard.png)

Revenue is seen to follow a predictable yearly cycle, with an average increase by a factor of 2.48 from Q1 to Q4. All three categories of products - Furniture, Office Supplies and Technology - are performing solidly with comparable revenue generated by each. The donut chart corresponding to the sale of office supplies displays only the top 5 highest generating sub-categories, and combines the rest into a slice labeled "Other", and updates this combination dynamcially based on filter and slicer selections made by the user. This was achieved using a constructed table (as can be seen in the semantic model) and a DAX measure which utilizes variables and a nested if statement. All of this creates a donut chart which is clear and minimizes unneccesary visual clutter.

![Profit - Dashboard](Dashboards/Profit%20-%20Dashboard.png)

The Canon imageCLASS 2200 Advanced Copier is the Superstore's best performing product, generating over three times the profit of any other item, while the furniture sub-category of tables is identified as a significant under-performer. A negative correlation is seen between the level of discount being applied and profit generated on individual sales, leading to the suggestion of a discounting upper limit of 40%.

![Revenue Breakdown - Dashboard](Dashboards/Shipping%20and%20Customer%20Segmentation%20-%20Dashboard.png)

The success rate of the "Same Day Delivery" shipping mode is high at an average of 96%, but as can be seen with the use of the interactive slicer on the dashboard, has in fact trended slightly downwards over the four years in question from 98% to 93%. As can be seen in the line graph, the average lead time across all orders does not appear to follow any yearly periodicity, showing solid order processing efficiency even in the quarters associated with substantially higher sales volumes.
